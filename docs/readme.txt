Read Me First
=============

XF Guestbook is a very complete guestbook :

- authorized for anonymous or not, with approval or not
- e-mail required or not
- possibility to add a photograph
- chose nation and display as flag
- chose gender and display as symbol
- display avatar and rank
You can disable or enable all these features.

- function for advanced preview
- form displayed by template
- send email to admin when new entry signed
- search function
- filter by gender, nation

Very complete administration
- lists of the messages (in standby/in progress)
- validation; edition, suppression of the messages
- management of the nations (upload flag,...)
